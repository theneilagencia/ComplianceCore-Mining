# QIVO Mining Design System

Bem-vindo à documentação completa do Design System da plataforma QIVO Mining.

## Visão Geral

Este Design System foi construído com **shadcn/ui** e **Tailwind CSS**, garantindo:

- **Consistência visual** em toda a plataforma
- **Acessibilidade WCAG 2.1 AA** em todos os componentes
- **Responsividade** mobile-first
- **Temas** claro e escuro
- **TypeScript** para type safety

---

## Paleta de Cores

### Cores Primárias

- **Primary (Roxo):** `#2f2c79` - Usado para CTAs principais e elementos de destaque
- **Secondary (Laranja):** `#b96e48` - Usado para acentos e elementos secundários

### Gradientes

- **Gradient Primary:** `linear-gradient(135deg, #2f2c79 0%, #b96e48 100%)`
- Usado em headers, cards de destaque e elementos visuais principais

### Cores Semânticas

- **Success:** Verde para ações bem-sucedidas
- **Warning:** Amarelo para avisos
- **Error:** Vermelho para erros
- **Info:** Azul para informações

## Componentes

### 53 Componentes Disponíveis

Organizados em categorias:

1. **Formulários:** Input, Textarea, Select, Checkbox, Radio, Switch
2. **Navegação:** Button, Link, Breadcrumb, Tabs, Pagination
3. **Feedback:** Alert, Toast, Dialog, Sheet, Drawer
4. **Layout:** Card, Separator, Skeleton, ScrollArea
5. **Dados:** Table, Badge, Avatar, Progress
6. **Overlays:** Popover, Tooltip, DropdownMenu, ContextMenu

---

## Acessibilidade

Todos os componentes seguem as diretrizes **WCAG 2.1 AA**:

- **Contraste de cores:** Mínimo 4.5:1 para texto normal
- **Navegação por teclado:** Tab, Shift+Tab, Enter, Escape
- **ARIA labels:** Todos os elementos interativos têm labels descritivos
- **Focus visible:** Outline visível em todos os elementos focáveis
- **Screen reader support:** Compatível com NVDA, JAWS, VoiceOver

---

## Como Usar

### Importar Componente

```tsx
import { Button } from '@/components/ui/button';

function MyComponent() {
  return <Button variant="default">Click me</Button>;
}
```

### Variantes

Cada componente tem múltiplas variantes:

```tsx
<Button variant="default">Default</Button>
<Button variant="destructive">Destructive</Button>
<Button variant="outline">Outline</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="ghost">Ghost</Button>
<Button variant="link">Link</Button>
```

### Tamanhos

```tsx
<Button size="default">Default</Button>
<Button size="sm">Small</Button>
<Button size="lg">Large</Button>
<Button size="icon">Icon</Button>
```

A plataforma inicia em **modo escuro** por padrão.

Usuários podem alternar entre claro e escuro usando o `ThemeToggle` component.
